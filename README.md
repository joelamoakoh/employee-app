  <h2>Introduction:</h2>
  <p>This is a basic employee management application that allows you to add, list, update, and delete employees, as well as generate pay slips and keep track of attendance. It is written in Kotlin and uses a simple UI for interaction.</p>
  <h2>Features:</h2>
  <ul>
    <li>Add Employees</li>
    <li>List Employees</li>
    <li>Search for Employees</li>
    <li>Generate Pay Slips</li>
    <li>Update Employee Information</li>
    <li>Delete Employees</li>
    <li>Track Attendance</li>
  </ul>
   <p>To run the application, simply clone the repository and open it in IntelliJ or any other Java development environment. You can then interact with the application using the UI, selecting the desired operation from the following menu:</p>
  <pre>
1 -> add()
2 -> list()
3 -> search()
4 -> paySlip()
5 -> update()
6 -> delete()
7 -> attendance()
-99 -> dummyData()
  </pre>
